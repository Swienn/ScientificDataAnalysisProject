# Scientific Data Analysis Project

Repository for preparing a municipality level dataset and analysis of the 2025 Dutch parliamentary election (TK2025).

## Purpose
This project combines multiple CBS and election CSV files to create a single municipality-level dataset (`Data/merged_tk2025_dataset.csv`) containing demographic variables, income statistics, migration background percentages, per-party vote shares, and bloc vote shares.

## Files of interest
- `MergeData.py` — main script that reads the source CSVs, computes per-gemeente statistics and vote shares, and writes `Data/merged_tk2025_dataset.csv`.
- `Data/` — contains source CSVs and output dataset. Key files used by `MergeData.py`:
  - `Inkomen_van_huishoudens__regio_11122025_135329.csv` — income per region (CBS)
  - `Regionale_kerncijfers_Nederland_11122025_133649.csv` — regional core figures (population)
  - `Kerncijfers_wijken_en_buurten_2024_11122025_141648.csv` — wijk/buurt data (used for migration background percent)
  - `uitslag_TK20251029_Gemeente.csv` — election results per gemeente
  - `PartyDistribution.csv` — party metadata (economic and migration blocs)
  - `merged_tk2025_dataset.csv` — output merged dataset (generated by `MergeData.py`)

## Requirements
- Python 3.8+ recommended
- Dependencies (install via pip):
  - pandas
  - numpy

You can install requirements with:

```bash
python -m pip install pandas numpy
```

## Usage
From the repository root run:

```bash
python MergeData.py
```

The script reads the CSVs from the `Data/` directory and writes `Data/merged_tk2025_dataset.csv`.

Notes
- The script contains robust CSV readers for CBS-style files (semicolon-separated, comma decimals).
- Municipality names are standardized before merging.
